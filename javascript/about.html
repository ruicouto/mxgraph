Component diagram editor for Alloy

1. Objective

The Alloy language supports the specification of architectures to be verified. Component diagrams are a kind of architecture in which it is possible to describe systems, as components, ports and connectors.

2. Instructions

The editor is composed of three main components. The canvas is located in the middle of the page, where the edition takes place. The canvas, on the left side, provides the elements that can be put in the canvas. Finally, the toolbar, provides additional features.

Components and containers can be dragged from the toolbar into the canvas. Ports can only be added to the components. Next, the ports can be connected to other ports, and to components.

Uploading a metamodel

It is possible to upload a metamodel in order to customise the canvas. A metamodel consists of an Alloy model, annotated so that the editor knows how to create the toolbar. An example of an Annotated Alloy model can be downloaded below.
